WowDash - ASP.NET Core 8 - Admin Dashboard Template
How to Install Admin Templates
1. Download and Extract the Zip File:

Download the zip file containing all files integrated with WowDash. Extract the contents of the zip file to your desired location.

2. Installing Visual Studio on Your Machine

To streamline development processes, ensure Visual Studio, a powerful integrated development environment (IDE), is installed on your machine. Download the latest version here. This IDE offers comprehensive tools for coding, debugging, and project management, enhancing productivity significantly.

3.Installing Node.js for SCSS Compilation

Ensure Node.js is installed on your system to generate CSS from SCSS. Node.js facilitates the use of npm packages like node-sass for efficient SCSS compilation. Download Node.js here to enhance your web development workflow with SCSS preprocessing.

4.Install the Sass Compiler

Once Node.js is installed, you can easily install the Sass compiler using npm. Run the below command in your terminal. This command installs the Sass compiler globally, making it accessible from anywhere on your system.npm install -g sass

5.Generate CSS using SASS compiler:

To generate CSS from SCSS, execute the command: npm run build If you want to run compiler in watch mode run: npm run dev

6.Opening Visual Studio:

Launch Visual Studio and navigate to File > Open > Project/Solution. Then, select the WowDash.sln solution file to open your project.

7.Running Application

Build and run your application in Visual Studio by pressing the F5 key. This will compile your code and launch the application for testing and debugging.
